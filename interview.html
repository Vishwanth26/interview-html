1.what is html ?

HTML, or HyperText Markup Language, is the standard language used to create and design documents on the web. It provides the structure for web pages, using a series of elements (often called tags) to define content such as headings, paragraphs, links, images, and other multimedia.

2.why we use html ?

HTML is essential for several reasons: Web Structure: It provides the basic structure for web pages, defining how content is organized and displayed. Content Formatting: HTML allows for formatting text, creating lists, tables, and sections, which helps in presenting information clearly. Linking: It enables the creation of hyperlinks, connecting different web pages and resources, facilitating easy navigation. Multimedia Integration: HTML supports embedding images, videos, and audio, making web pages more engaging and interactive. Accessibility: Proper use of HTML elements enhances accessibility for people using screen readers and other assistive technologies. SEO Benefits: Search engines use HTML structure to index content effectively, which can improve a site's visibility in search results. Cross-Platform Compatibility: HTML is universally supported by all web browsers, ensuring that web pages can be viewed consistently across different devices and platforms. Foundation for Other Technologies: HTML works alongside CSS (for styling) and JavaScript (for interactivity), forming the core technologies of web development.

9.What is d/f b/w the inline vs block elements?

The key differences between inline and block-level elements in HTML are: 1. Layout Behavior: Block-level elements: Always start on a new line and take up the full width available (from left to right in the container). Inline elements: Do not start on a new line and only take up as much width as their content requires. 2. Content Flow: Block-level elements: Stack vertically, one on top of the other. Inline elements: Flow horizontally within the block they are contained in, side by side. 3. Nested Elements: Block-level elements: Can contain other block-level or inline elements. Inline elements: Typically can only contain other inline elements or text (block elements inside inline elements are invalid). 4. Width and Height: Block-level elements: You can explicitly set their width, height, margin, and padding. Inline elements: Width and height cannot be set directly (they are based on the content), but padding and margin can affect them.
